2. 配置 iTerm2 主题

iTerm2 最常用的主题是 Solarized Dark theme.

下载地址：http://ethanschoonover.com/solarized

  1. 解压下载的压缩文件；

  2. 打开 iTerm2，使用快键键 Command + , 键，或者 iTerm2 -> Preferences 打开Preferences配置界面；

  3. 按照 Profiles -> Colors -> Color Presets -> Import操作，导入刚刚下载解压的主题文件：solarized->iterm2-colors-solarized->Solarized Dark.itermcolors文件；

  4. 导入成功，选择 Solarized Dark 主题。
