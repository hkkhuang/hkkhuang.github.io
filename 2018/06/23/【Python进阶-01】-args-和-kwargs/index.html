*args的用法

*args 和**kwargs 主要用于函数定义 可以将不定数量的参数传递给一个函数。

注释: 不定的意思是，事先不清楚函数的使用者会传递多少参数给函数。

*args 用来发送一个非键值对的可变数量的参数列表给一个函数。

**kwargs的用法

**kwargs 用来发送不定长度的键值对作为参数传递给一个函数。

使用 *args 和 **kwargs 来调用函数

不是必须写成*args 和**kwargs。 只有变量前面的 *(星号)才是必须的。
也可以写成*var 和**vars。
